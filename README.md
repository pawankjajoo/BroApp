# BroApp

Social networking app built with React Native/Expo and Firebase.

## Core Features

- Bros - friend roster with messaging
- Bro-cast - broadcast messages to all frends
- Bro-ximity - location-based friend discovery
- Bro-mmunity - global activity feed
- Wallet - in-app currency with IAP support
- Bro-file - user profile with BroCoin rewards

## BroCoin System

- Token minting with proof-of-work
- Peer-to-peer transfers with 30% platform fee
- Public ledger tracking
- Anti-bot measures via proof-of-work challenges and behavior event tracking

## Architecture

- 11 screens, 8 service modules
- Firebase backend: Firestore, Storage, Authentication (email + OAuth)
- Push notifications via Expo
- In-app purchases via expo-in-app-purchases
- Proof-of-work anti-bot protection

## Tech Stack

- React Native, Expo SDK
- Firebase (Auth/Firestore/Storage)
- expo-location, expo-notifications, expo-in-app-purchases

Development - core architecture complete, some service modules have incomplete implementations
