# Bro Launch Plan

Social app for your actual friend group. Free with Bro Bucks currency. iOS + Android via React Native/Expo.

## Phase 1: Finish the App

- [ ] All six tabs working (Bros, Bro-cast, Bro-ximity, Bro-mmunity, Bro Bucks, Bro-file)
- [ ] Chat with push notifications
- [ ] Bro-cast feed (photos, text, reactions)
- [ ] Bro-ximity friend finding with privacy controls
- [ ] Bro Bucks wallet: earn and send
- [ ] Community creation and discovery
- [ ] Profile setup
- [ ] Auth flows (signup, login, reset)

## Phase 2: Friends and Family Beta

- [ ] TestFlight + Google Play internal track
- [ ] 50 testers across a few friend groups
- [ ] See if people actually use it daily
- [ ] Test message delivery and notification timing
- [ ] Test Bro-ximity accuracy in different spots
- [ ] How fast can someone get their crew on the app?
- [ ] Fix bugs and friction

## Phase 3: App Store Submission

- [ ] Screenshots and preview video
- [ ] App Store descriptions
- [ ] Privacy policy and ToS
- [ ] Content moderation plan
- [ ] Submit to both stores
- [ ] Crash reporting and basic analytics

## Phase 4: Campus Launch

- [ ] Start at 1-2 campuses where you know people
- [ ] Campus ambassadors (5 per school)
- [ ] Instagram/TikTok presence
- [ ] Referral: invite 5 bros, get Bro Bucks
- [ ] Track if users invite friends on their own
- [ ] Watch what features get used vs ignored

## Phase 5: Grow

- [ ] 10+ campuses
- [ ] Premium features (themes, enhanced profile)
- [ ] Bro Bucks purchase packs
- [ ] Group chats and group Bro-casts
- [ ] Events in Bro-mmunity

## Targets

| | Beta | Campus Launch | 6 Month |
|-|------|---------------|---------|
| Users | 50 | 500 | 5,000 |
| DAU/MAU | - | 40% | 50% |
| Referral rate | - | 2/user | 3/user |
